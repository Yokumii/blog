卡尔曼滤波

2024/10/26 数据挖掘与分析 数据挖掘与分析

1. 简述

在目标轨迹追踪领域，传统均值/中值滤波器仅能处理位置信息且存在滞后问题。R.E.Kalman于1960年提出的卡尔曼滤波，通过融合物理动力学模型与传感器观测数据，实现状态向量（位置、速度等）的最优估计。其核心优势在于：

  • 利用先验知识降低测量噪声影响
  • 无延迟的实时递推计算
  • 提供估计结果的置信度评价（协方差矩阵）

3. 应用与参数调优

3.1 过程噪声影响

以行人轨迹跟踪为例，过程噪声协方差$Q$中的$\sigma_s$参数调节动力学模型信任度：

  • 大$\sigma_s$值：优先响应观测数据，轨迹更接近原始测量
  • 小$\sigma_s$值：强化物理约束，输出更平滑轨迹

3.2 Python实现示例

import numpy as np

# 状态向量 [位置, 速度]
x = np.array([[0.0], [0.0]])
F = np.array([[1, 1], [0, 1]])  # 状态转移矩阵
H = np.array([[1, 0]])          # 观测矩阵
Q = np.diag([0.1, 0.01])       # 过程噪声协方差 
R = np.array([[10.0]])         # 测量噪声方差

for measurement in measurements:
    # 预测步骤
    x = F @ x
    P = F @ P @ F.T + Q

    # 更新步骤
    K = P @ H.T @ np.linalg.inv(H @ P @ H.T + R)
    x += K @ (measurement - H @ x)
    P = (np.eye(2) - K @ H) @ P

4. 总结

4.1 优势

  • 线性高斯系统下的最优估计
  • 计算复杂度仅与状态维度立方相关
  • 提供实时置信度评分
